# AI Love Quote Generator

A beautiful and romantic AI Love Quote Generator built with HTML, CSS, and JavaScript.

## Features

- **Romantic UI:** A soft, dreamy romantic theme with shades of light pink, lavender, and white.
- **Dynamic Quotes:** Fetches love quotes from the [quotable.io](https://quotable.io/) API.
- **Fallback System:** If the API fails, it displays a quote from a predefined list.
- **Animations:** Smooth animations for a cozy and elegant feel.
- **Responsive Design:** Looks great on all screen sizes.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
